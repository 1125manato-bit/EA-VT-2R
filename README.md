# EA VT-2R

Aggressive Saturation Plugin by EMU AUDIO.

## Character

- **Punchy / Aggressive / Forward**
- Circuit: Transformer + Solid State

## Parameters

- **DRIVE (0-100)**: Controls saturation intensity. Boosts mid-frequencies (2kHz) before saturation for a "forward" character.
- **MIX (0-100)**: Dry/Wet blend.

## Build

### macOS

```bash
./build.sh Release
```

### Windows

```cmd
build.bat
```
